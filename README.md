# 📖 About

**BLACK THREAD** is an original psychological manga that explores fear, forgotten memories, destiny, and the darkness hidden within the human mind.

Every choice creates a thread.

Every thread leads somewhere.

Some lead to salvation...

Others lead to the abyss.

---

# 🌌 Synopsis

Room 404.

A place that should not exist.

A place where forgotten memories awaken.

As reality begins to collapse, the line between nightmares and truth slowly disappears.

The deeper the protagonist searches for answers...

...the closer he gets to a truth that was never meant to be discovered.
